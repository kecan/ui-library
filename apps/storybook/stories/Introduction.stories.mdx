# 🎨 KC UI 组件库

欢迎使用 KC UI 组件库！这是一个基于 React + TypeScript 构建的现代化组件库，采用 monorepo 架构管理，为你的应用提供高质量、可复用的 UI 组件。

## ✨ 特性

- **🔧 TypeScript 优先** - 完整的类型定义，提供出色的开发体验
- **🎨 现代设计** - 简洁美观的视觉效果，符合现代设计趋势
- **📱 响应式设计** - 完美适配桌面端和移动端
- **♿ 无障碍友好** - 遵循 WCAG 标准，支持键盘导航和屏幕阅读器
- **🔄 高度可定制** - 灵活的主题系统，支持深度定制
- **📦 按需加载** - 支持 Tree Shaking，减小打包体积
- **⚡ 性能优化** - 轻量级实现，注重性能表现

## 📦 安装

```bash
# 使用 npm
npm install @kc-ui/components

# 使用 pnpm
pnpm add @kc-ui/components

# 使用 yarn
yarn add @kc-ui/components


import React from 'react'
import { Button, Input } from '@kc-ui/components'

function App() {
  return (
    <div>
      <Button variant="primary" size="medium">
        开始使用
      </Button>
      
      <Input 
        placeholder="请输入内容..." 
        type="text"
      />
    </div>
  )
}

export default App


// 在你的入口文件中引入样式
import '@kc-ui/components/dist/index.css'
